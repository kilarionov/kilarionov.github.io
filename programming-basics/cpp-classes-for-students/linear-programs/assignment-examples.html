ПРИМЕРИ за програми с Линейна Структура

Пример № 1
Да се пресметне и отпечата периметъра P на квадрат със страна а.

#include <iostream>
using namespace std;
main ()
{
	double a, P;
	cin >>a;
	P=4*a;
	cout <<P;
}
Изход

Пример № 2

Да се пресметне и отпечата лицето S и периметъра P на квадрат със страна а. 

#include <iostream>
using namespace std;
main ()
{
	double a, S, P ;
	cin >>a;
	S=a*a;
	P=4*a;
	cout <<S<<"\t"<<P;
}
Изход

Пример № 3

Да се пресметне и отпечата последната цифра на зададено цяло число k. Примерно ако к е 3254 да се отпечата 4. 

#include <iostream>
using namespace std;
main ()
{
	int k, lastDigit ;
	cin >>k;
	lastDigit = k%10;
	cout <<lastDigit;
}
Изход

Пример № 4

Да се пресметне и отпечата предпоследната цифра на зададено цяло число k. Примерно ако к е 3254 да се отпечата 5. 

#include <iostream>
using namespace std;
main ()
{
	int k, noLastDigit, lastDigit ;
	cin >>k;
	noLastDigit = k / 10;
	lastDigit = noLastDigit % 10;
	cout <<lastDigit;
}
